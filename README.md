# MicroPython ST7920
A MicroPython driver for the ST7920 128x64 LCD Display.

![Example Image Of Display](example.webp)

## Features
- Communicate over SPI
- Supports MicroPython FrameBuffers

## Works On
- Raspberry Pi Pico
